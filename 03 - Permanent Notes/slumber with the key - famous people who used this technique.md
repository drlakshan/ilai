# reference 

Dali employed a method he described as '**slumber with a key**' which involved him sitting in a favourite bony Spanish styled armchair while holding a heavy key over an upside-down plate. Dali would think of issues he wanted to explore, using Focused Thinking, then let himself fall asleep as he pondered them
