######  Note


Type A : middle ear
Type B : middle ear and mastoid without infralabyrinthine spread
Type C
	C1 : infralabyrinthine with vertical canal of carotd encroached
	C2 : infralabyrinthine with vertical canal infiltrated
	C3 : infralabyrinthine with horizontal caratid canal infiltrated
Type D
	D1 : intracranial extension limited <2 cm
	D2 : intracranial extension > 2 cm
