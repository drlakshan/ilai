# DeMeester and Jonson Criteria for Reflux
2022-03-04- 08:34


## Note

Set of criteria used for diagnosing occult reflux
Need ambulatory pH monitoring

6 variables are used
	1. number of reflux episodes
	2. number of episodes lasting more than 5 minutes
	3. maximal reflux duration
	4. total percentage of time where pH was below 4 for the total monitoring
	5. percentage of time where pH below 4 in an upright position
	6.  percentage of time where pH below 4 in supine position
