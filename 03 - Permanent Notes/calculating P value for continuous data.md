2022-06-29- 09:12


######  Note

P value is what is the probability the observed difference is "real" 95% with confidence
avoiding [[type 1 statistical  error]] of false positive labelling when actually there isn't a difference
to less than 5%

three tests

1. Student's T test
2. Mann-Whitney Test
3. ANOVA test
