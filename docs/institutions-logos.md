# Institution logos

Add logo files to:

`public/images/institutions/`

The institutions page will automatically:
- show the official logo if the file exists and loads
- fall back to the monogram badge if it does not

## Recommended asset style

- transparent PNG preferred
- SVG is excellent when available
- WEBP is fine when the source is clean
- square or near-square where possible
- keep background clean
- simple monochrome or high-contrast marks work best
