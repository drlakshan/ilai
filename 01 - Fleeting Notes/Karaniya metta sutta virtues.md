###### Karaniya metta sutta virtues
2022-09-10- 17:18





######  Note

CONTEXT:

When I read about the Benjamin Franklin's virtues, it reminded me about Metta sutta virtues

NOTE:

1. Sakko : Daksha - Able
2. Ujucha : ridgu  - Sincere, Just particularly not been shaken by sathara agathiya 
3. Sujjucha : Mana kota ridgu candid to yourself 
4. Suwachochassa : keekaru : obidient 
6. Mudu : gentle
7. Anathimani : nihathamani : humble
8. Santhussakocha : lada deyin sathutu wanneya - frugality
9. Subarocha : pahasuwen poshanaya kala haki
10. Appakichchocha : swalpa katayuthu aththeki
11. Sallawakuththi : sahallu pawathum aththeki
12. Santhridayocha : sansun iduran aththeki
13. Nipakocha : nuwana athttheki
14. Appagabbo : hadi dadi asanwara gathi naththeki : tranquility
15. Kulesu annunugiddo : kulayanhi no elenneya
16. Nacha Kuddan Samachare Kinchiyeno pare ananuguddidan : Gnanawanthayan wisin nigraha karana sulu waradakwath nokara sitinneya
